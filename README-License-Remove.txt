1) ABOUT ORCHEVO
==================================================

==================================================
3) HOW TO OPEN ORCHEVO ON MACOS
==================================================

If macOS shows a security message because the app is not signed with an Apple Developer account:

1. Try opening Orchevo normally once.
2. Open System Settings.
3. Go to Privacy & Security.
4. Look for the blocked app message.
5. Choose Open Anyway for Orchevo if needed.
