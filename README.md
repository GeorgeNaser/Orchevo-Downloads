## Installation on macOS

1. Download `Orchevo.dmg`
2. Open the DMG
3. Run `Orchevo.pkg`
4. Follow the installer steps

If macOS shows a security warning:
- Open **System Settings**
- Go to **Privacy & Security**
- Choose **Open Anyway** for Orchevo if needed
